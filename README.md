Iris Flower Classification

This project uses the famous Iris dataset to build a machine learning model that classifies iris flowers into different species based on their physical measurements. The dataset includes features such as sepal length, sepal width, petal length, and petal width, which are used to predict the flower species.

The project follows a complete machine learning workflow, including data loading, preprocessing, label encoding, model training, and evaluation. A Support Vector Machine (SVM) classifier with a linear kernel is used to classify the flowers, and the model’s performance is assessed using a classification report and confusion matrix. The results are also visualized to compare actual and predicted species distribution.

This project is a simple yet effective example of classification in machine learning and demonstrates how Python libraries such as pandas, seaborn, matplotlib, and scikit-learn can be used together to solve a real problem.
Key Features

Data loading and exploration
Label encoding for species
Feature selection and train-test splitting
SVM model training and evaluation
Confusion matrix and classification report
Visualization of prediction results
